# Dr. R. Jordan Price - Personal Website

This repository contains the source code for my personal academic website and blog, hosted at [rj-price.github.io](https://rj-price.github.io).

## About Me
I am a Bioinformatician and Molecular Biologist. My work focuses on:
- Bioinformatics & Computational Biology
- Functional Genomics
- Molecular Biology

## Features
The site is built using [Jekyll](https://jekyllrb.com/) and the [al-folio](https://github.com/alshedivat/al-folio) theme. It includes:
- **CV**: A comprehensive overview of my academic and professional background.
- **Publications**: Automatically generated from BibTeX.
- **Projects**: Highlights of my research and software projects.
- **Blog**: "Getting Started with Bioinformatics" - a collection of guides and tutorials.

## Local Development
To run the site locally, you need to have Ruby and Jekyll installed:

```bash
bundle install
bundle exec jekyll serve
```
